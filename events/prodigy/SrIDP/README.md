# Senior Industry Defined Problem (Sr. IDP) – Vortex 13.0

This repository contains the official documents for the **Senior Industry Defined Problem (Sr. IDP)** event conducted as part of **Vortex 13.0**, the annual chemical engineering festival of the **Institute of Chemical Technology (ICT), Mumbai**.

Sr. IDP is a flagship event where participants work on **real-world problem statements defined by industry partners**, requiring technically sound, feasible, and innovative solutions.

---

## Contents

* **Problem Statements 2025**
  Industry-defined problem statements (IDP A–F) covering process design, separations, pharmaceuticals, sustainability, reaction engineering, and waste treatment.

* **Event Overview Document**
  Official introduction to Sr. IDP, including event structure, evaluation format, industry participation, and sponsorship details.
